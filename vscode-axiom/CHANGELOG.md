# Changelog

All notable changes to the Axiom VS Code extension will be documented in this file.

## [0.1.0] - 2026-04-09

### Added
- Initial release
- Syntax highlighting for `.axiom` spec files
- Language Server Protocol (LSP) integration
  - Real-time diagnostics
  - Auto-completion
  - Hover documentation
  - Go to definition
- Commands:
  - Build Current Spec
  - Verify Current Spec
  - Restart Language Server
- Status bar indicator
